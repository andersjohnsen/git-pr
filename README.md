# git-pr
Git utility for working with GitHub Pull Requests

## Install

To install put the tool into your path, e.g.:

```
$ ln -s `pwd`/git-pr ~/.local/bin/git-pr
```

The tool can now be invoked by saying

```
$ git-pr
Usage: git-pr [OPTIONS] COMMAND [ARGS]...
[...]
```
